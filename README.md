# Dynamic Web Project (Java Web Application)

This project is a **Dynamic Web Application** developed using **Java** in **NetBeans IDE**.  
It demonstrates core concepts of Java web development, including project structure, configuration files, and deployment-ready build outputs.

The project follows a standard NetBeans dynamic web project layout and is suitable for learning, academic use, and basic web system development.
